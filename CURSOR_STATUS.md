## 📋 规则（遵守，别问为什么）

1. **生肖只认岁数法**：全站没有固定生肖表，严禁写 01=鼠 02=牛 这种
2. **岁数法公式**：号码÷12=余数，太岁=余1，反向排列
3. **双语纯净化**：中文页不能有英文残留，英文页不能有中文残留
4. **号码输出**：得特肖→号码=该肖岁数+12N。如狗=9岁→9/21/33/45
5. **首页双卡片不动**：布局、动画、轮询逻辑全部完成
6. **国际卡 7 字圈动画**：一字一圈，21:20→21:28→21:30 三段切换
7. **后台权限**：/admin 受 guard 保护，login 页面豁免
8. **API 缓存**：公共 API 统一 force-dynamic，不缓存
9. **日期可编辑、数字输入去箭头、本页说明全删**——这些都是主人明确要求的
10. **修改范围**：只改主人说的，别顺手"优化"已完成的东西
